# Install `node`

## 1) First install Node

### Mac

```bash
brew install node
```

### Windows

```bash
curl -sL "https://deb.nodesource.com/setup_12.x" | sudo -E bash -
sudo apt-get install -y gcc make ruby ruby-dev
sudo apt-get install -y nodejs
```

## 2) Now check Node installed

Run `node -v`

We expect something like:

```bash
v12.16.2
```
